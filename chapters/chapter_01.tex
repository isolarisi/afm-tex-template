Использование горячих клавиш для быстрой вставки тегов.

Для использования скриптов необходимо:
\begin{itemize}
\item скопировать файлы из {\sl scripts} в папку{\sl C:$\backslash$Users$\backslash$<имя пользователя компьютера>$\backslash$AppData$\backslash$Local$\backslash$MiKTeX$\backslash$2.9$\backslash$TeXworks$\backslash$0.6$\backslash$scripts$\backslash$LaTeX styles};
\item в меню TeXworks: {\sl Scripts $\Rightarrow$ Scripting TeXworks $\Rightarrow$ Reload Script List}.
\end{itemize}
